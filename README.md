# Linear Algebra Lab

This repository is for learning linear algebra by building small code examples.

## Topics
- Vectors
- Matrices
- Linear transformations
- Dot products
- Projections
- Eigenvalues and eigenvectors

## Goal
Understand linear algebra deeply enough to use it in robotics, 3D vision, and machine learning.

## Rule
Every concept must be:
1. studied
2. written in notes
3. implemented in code
4. tested with an example
